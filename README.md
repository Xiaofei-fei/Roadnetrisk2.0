# 项目介绍
本项目是城市交通状态与安全态势协同关联分析平台，可以对不同路段的交通态势进行查看，对其发生事故的指标进行分析。

---

更新：新增加了路况一天内的动态变化情况，在指定路段上点击事故图标即可显示。涉及到的路段包括10，37，61，163号

该项目可通过网址：https://roadnetplat.netlify.app/  进行查看。
